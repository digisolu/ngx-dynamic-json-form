# Global Configurations

A lot of default configuration parameters can be set global, so there is no need to pass them every time with your JSON.

## Example

```typescript
@NgModule({
  declarations: [
    AppComponent,
    MyCustomFormFieldComponent,

    /** Other declarations */
  ],
  imports: [
    /** Other imports */

    NgxDynamicJsonFormMaterialModule.forRoot({
      components: {
        "my-custom-form-field": MyCustomFormFieldComponent,
      },
      layoutOptions: {
        default: {
          appearance: "outline",
          floatLabel: "always",
        },
        select: {
          showSearch: false,
        },
      },
    }),

    /** Other imports */
  ],
  providers: [
    /** Other providers */
  ],
  bootstrap: [AppComponent],
})
export class AppModule {}
```
